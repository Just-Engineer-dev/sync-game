## Requirements

## Common setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/TheMeekrok/sync-game.git
cd sync-game.git
```

```bash
npm init
npm install --save express socket.io
```

Open [http://localhost:3000](http://localhost:3000) and take a look around.
